# Discord Quest Finisher

Discord Quest Finisher is a Node.js CLI tool that automates Discord Quests completion via Discord API v10.

---

## Features

- Automatically completes Discord quests
- Supported all task types
- Sequential mode (safer)
- Parallel mode (faster, higher risk)
- Complete a single specific quest
- Automatic retries and progress tracking

---

## Requirements

- Node.js v18+
- Discord user token
- Android or Linux or Windows or MacOS device
---

## Installation & Usage

```bash
git clone https://github.com/xm5o/discord-quest-finisher.git
cd discord-quest-finisher
node index.js
```

Set your token in the main file:

```js
const TOKEN = 'PUT_YOUR_DISCORD_TOKEN_HERE';
```
